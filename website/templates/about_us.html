About Us

Predict, Analyze, and Gain Insights from Market Data and News

Stock Market Analysis is a web application built as a Final Year Project. It combines a numerical machine learning model for technical analysis with a FinBERT-based Natural Language Processing model for sentiment analysis. Together, these models provide valuable insights into stock performance and market reactions to news. The application is developed using the Flask framework and designed with HTML, CSS, and JavaScript.
